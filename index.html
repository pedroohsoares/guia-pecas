Material de Apoio para Capacitação em Robótica Educacional

Projeto 1: Robô Desviador de Obstáculos

Arduino Uno

Função Principal: É o cérebro do robô. Executa o código, lê os sensores e comanda os motores, tomando todas as decisões lógicas.

Protoboard

Função Principal: A base de montagem. Permite conectar componentes eletrônicos de forma temporária e segura, sem necessidade de solda.

Sensor Ultrassônico

Função Principal: Os olhos do robô. Emite ondas sonoras e mede o tempo do eco para calcular com precisão a distância de objetos.

Micro Servo Motor

Função Principal: O pescoço do robô. Gira para ângulos específicos para que o sensor possa "olhar" para os lados e procurar rotas de fuga.

Driver de Motor L293D

Função Principal: O controlador dos motores. Recebe os sinais do Arduino e gerencia a energia de uma fonte externa para controlar a direção (frente/ré) e a velocidade.

Motores DC com Rodas

Função Principal: As pernas e rodas. Convertem energia elétrica em movimento, permitindo que o robô se desloque pelo ambiente.

Jumpers Macho-Macho

Função Principal: Conectar dois pontos com entradas fêmeas. São usados para ligar pontos na protoboard entre si ou para conectar a protoboard aos pinos do Arduino.

Jumpers Macho-Fêmea

Função Principal: Conectar um pino macho a uma entrada fêmea. São essenciais para ligar componentes que já têm pinos (como o Sensor Ultrassônico) diretamente ao Arduino ou à protoboard.

Projeto 2: Semáforo Inteligente

LEDs

Função Principal: Os sinalizadores visuais. O Arduino acende a cor correspondente (verde, amarelo, vermelho) para indicar o nível de proximidade detectado.

Resistores

Função Principal: Os protetores dos LEDs. Limitam a corrente elétrica para garantir que o LED receba apenas a energia necessária para acender, sem queimar.
